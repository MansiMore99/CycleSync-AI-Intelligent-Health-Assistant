### CycleSync AI

A Next.js-powered wellness assistant that leverages AI to provide personalized recommendations for workouts, nutrition, and self-care based on menstrual cycle phases. Featuring a chatbot built with Quadrant, a graph database for symptom mapping, and an interactive user interface, CycleSync AI helps users align their lifestyle with their body's natural rhythm. ðŸš€
